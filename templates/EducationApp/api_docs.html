احراز هویت

برای استفاده از متدهای POST، PUT، PATCH و DELETE نیاز به توکن احراز هویت دارید. توکن را می‌توانید از آدرس زیر دریافت کنید:

POST /api/token-auth/ Body: {"username": "yourusername", "password": "yourpassword"}

توکن دریافتی را در هدر درخواست‌های خود به صورت زیر قرار دهید:

Authorization: Token your_token_here
